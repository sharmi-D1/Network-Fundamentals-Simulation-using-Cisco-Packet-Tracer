📌 Overview

This project focused on simulating and analyzing network communication using Cisco Packet Tracer to better understand how network devices such as hubs and switches function within a network environment.

The project involved building network topologies, observing packet flow, analyzing traffic behavior, and comparing the performance and functionality of hub-based and switch-based networks. This hands-on experience provided practical exposure to networking concepts, device communication, and traffic forwarding mechanisms used in real-world networks.
